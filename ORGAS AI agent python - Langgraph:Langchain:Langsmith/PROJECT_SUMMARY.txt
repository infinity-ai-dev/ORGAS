🚀 COMO COMEÇAR
═══════════════════════════════════════════════════════════════════════════

1. ENTRAR NO DIRETÓRIO
   $ cd "ORGAS AI agent python - Langgraph:Langchain:Langsmith/backend"

2. INSTALAR DEPENDÊNCIAS (primeira vez)
   $ uv pip install -r requirements.txt

3. VERIFICAR TUDO ESTÁ OK
   $ DEBUG=true uv run python test_integration.py
   Output: ✅ TODOS OS TESTES PASSARAM!

4. INICIAR SERVIDOR
   $ DEBUG=true uv run python main.py
   Output: INFO: Uvicorn running on http://0.0.0.0:8000

5. TESTAR EM OUTRO TERMINAL
   $ curl -X POST http://localhost:8000/analyze \
     -H "Content-Type: application/json" \
     -d '{"filename": "test.pdf", "content": "NOTA FISCAL..."}'
